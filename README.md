# SafePlace

Mobile Systems (Study Subject)

## Instruction to run project:

1. Clone project.
   ```javascript
   git clone https://github.com/Phabes/SafePlace.git
   ```
1. Open "safePlace" folder in command line.
   ```javascript
   cd safePlace
   ```
1. Run:
   ```javascript
   npm install
   ```
1. Run:
   ```javascript
   npm start
   ```
